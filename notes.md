## steps

- create repo
- clone repo
- cd into folder
- `npm init -y`
- `npx gitignore node` or `npm -i gitignore` and then `gitignore node`
- `git add .`
- `git commit -m "message"`
- `git push -u origin master`

Express API are broken down into Express Routers.
